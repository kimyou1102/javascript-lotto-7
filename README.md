# javascript-lotto-precourse

## 구현 기능 목록

- [ ] 로또 구입 금액을 입력받는다.
- [ ] 당첨번호는 중복되지 않는 숫자 6개를 입력받는다.
- [ ] 보너스 번호 1개를 입력받는다.
  - [ ] 당첨번호와 중복되지 않는 숫자여야 한다.
- [ ] 발행한 로또 수량 및 번호를 출력한다.

  - [ ] 구입금액 / 1000 개 만큼 로또를 발행한다.

- [ ] 사용자가 구매한 로또 번호와 당첨번호를 비교해 당첨 내역을 출력한다.
- [ ] 수익률을 계산하고 출력한다.

## 로또

- [ ] 숫자 범위는 1~45
- [ ] 중복되지 않는 6개의 숫자를 가진다.

### 입력

- [ ] 로또 구입 금액 입력받는다.
- [ ] 당첨 번호 입력받는다.
- [ ] 보너스 번호 입력받는다.

### 출력

- [ ] 발행한 로또 목록
- [ ] 당첨 통계
- [ ] 수익률
- [ ] 에러 발생시 [ERROR]로 시작하는 에러문구
